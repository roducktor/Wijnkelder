MijnKelder
Verbind je gratis Supabase database om je wijnkelder op te slaan en te synchroniseren tussen al je apparaten.

Eenmalige setup (5 min)

1

Maak een gratis account op supabase.com en maak een nieuw project aan.

2

Ga naar SQL Editor en plak dit erin:
CREATE TABLE wines (id uuid DEFAULT gen_random_uuid() PRIMARY KEY, name text, type text, year int, producer text, region text, bottles int DEFAULT 1, price numeric, drink_from int, drink_to int, rating int, notes text, ai_info text, created_at timestamptz DEFAULT now());

3

Ga naar Project Settings → API en kopieer je Project URL en anon public key.

Je gegevens worden alleen lokaal opgeslagen in deze browser om verbinding te maken met jouw database.
